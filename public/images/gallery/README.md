# Gallery Images

This directory contains gallery images for the UIT portal.

## Image Requirements

- **Format**: JPG, PNG, WebP
- **Size**: Maximum 2MB per file
- **Dimensions**: Minimum 800x600px, optimal 1200x800px
- **Quality**: 85% compression for balance between file size and quality

## Folder Structure

```
gallery/
├── 2024/
│   ├── 01/ (January)
│   ├── 02/ (February)
│   └── 03/ (March)
├── prestasi/
├── fasilitas/
└── kegiatan/
```
